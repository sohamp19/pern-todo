# pern-todo

A simple PERN (Postgres Express React Node) todo list

To run in on your own local machine,

1. ```
   git clone https://github.com/sohamp19/pern-todo
   ```
2. ```
   cd server
   ```
3. ```
   node index
   ```
4. ```
   cd ../client
   ```
5. ```
   npm run build
   ```
6. ```
   npm run start
   ```
